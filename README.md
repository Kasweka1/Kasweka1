A REST API for managing library operations — books, members, loans, returns. The longer-term vision is removing the dependency on expensive barcode scanners by letting librarians scan books through a mobile app instead. Work in progress.

A learning management system built with the intention that Zambian schools with a developer can fork and self-host it for free. Aligns with my research interest in accessible, locally-relevant education infrastructure.

A found-and-lost system built for a friend. Simple, functional, and a good example of how I approach real-world problem solving over tutorial-style projects.
